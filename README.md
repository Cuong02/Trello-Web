# Trello Web

## Requirements

- Node v18.16.0
- Git

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/Cuong02/Trello-Web.git
```

```bash
npm install
```

Open Config & rename .env.example to .env

## Command

> npm run clean

> npm run build-babel

> npm run build

> npm run production

> npm run dev
